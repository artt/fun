Each point is (X, f(X)). Because f ignores digit order and every output is a multiple of 9, thousands of inputs collapse onto a few output levels — the horizontal bands. Zoom in and you'll see each band is a dense row of distinct X's that all land on the same value. In a trace, white segments rise from X up to f(X); rose segments slide across to the 45° line, where the next step begins.
